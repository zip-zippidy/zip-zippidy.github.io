Our Work

This page documents all the development prompts and requests that shaped this website. Below is a chronological list of all instructions provided during the project development.

Project Development Prompts

  1. Initial Analysis: "Can you recognize web page contents and explain where does index.html gets its background?"
  2. Background Removal: "Remove all background"
  3. Background Color Change: "make all background in pink color"
  4. Link Removal: "Find all links to other locations, remove links and any mention of them from index page"
  5. Enterprise Creation: "Generate fictional Enterprise with Title: Web page development with AI. Change page content to include all needed information of normal webpage. Site should be associated with title name and should contain some made up but related information about AI made web page development as a service. Add small description about this company"
  6. Company Location & Pricing: "Change company location to Estonia, Tallinn at TalTech university and remove all satellite studios. Remove all buttons and links created. Change currency of cost per project to euro and cost starting from 3k euro."
  7. New Pages Creation: "Create two pages. One is Contacts. Which will maintain same formatting but will be left without content. Second is Our work. Which will maintain same formatting but will be left without content."
  8. Navigation Links: "Create buttons and links to those pages."
  9. Button Styling & Typography: "Format buttons to appear more visible and make their appearance as actual interactive buttons. Change all text format to any other. Change colour of text 25% darker if not black."
  10. Button Enhancement & Images: "Change button appearance to 20% transparent, change colour to dark pink and align them in the middle, make all buttons in same size and increase button width to be 15% wider. Before buttons add three pictures of AI generating web pages and make animations of those pictures changing after each 2 seconds."
  11. Template Images & Contact Info: "Change pictures to actual pictures of webpage templates. Add welcoming greeting in contacts page and add linkedin profile contact and email. LinkedIn https://www.linkedin.com/in/radions-l-525469299 and email radion@taltech.ee."
  12. Documentation Request: "In our work page include all prompts that I gave you list them as list and format for better readability. Also include this last prompt."

Project Summary

This website was developed through a series of iterative prompts, transforming from a basic template into a fully customized AI web development company showcase. The project includes three main pages (Home, Our Work, and Contacts), custom styling, interactive elements, and animated webpage template previews.

Web page development with AI is based in Tallinn, Estonia at TalTech University. We help product marketers, founders, and creative directors launch unforgettable web experiences by pairing human insight with machine intelligence.
